GitHub快速上手

开始阅读

  • Step1:注册
    • 打开浏览器
    • 进入官网：https://github.com/
    • 点击Sign up注册
  • Step2:新建项目
    • 点击New Repository开始新建项目
  • Step3:填写项目信息
    • 填写Repository name、Description
    • Readme建议勾选 lience指需要遵从的协议
    • 点击Create Repository完成项目创建
  • Step4:项目信息展示
    • 箭头指向为上传/下载地址
  • Step5:本地安装git
    • 安装包下载路径http://pan.baidu.com/s/1dE4NMhz
    • 安装默认安装在C盘，装在C盘可能造成权限不够无法读写的情况，所以建议安装在其他盘。
  • Step6:右击Git文件夹，运行Git Bash Here
    • 输入git clone 加上上述红箭头指向的加载路径将项目克隆到本地
    • 克隆完成后就可以在Git文件夹下找到项目文件了
  • Step7:提交修改文件
    • 修改，下面执行提交修改操作
    • cd cloudsails //进入到这个文件夹
    • git add . //将改动的地方添加到版本管理器
    • git commit -m "some changes" //提交到本地的版本控制库里，引号里面是你对本次提交的说明信息
    • git push -u origin master //将你本地的仓库提交到你的github账号里，首次提交需要输入你的github的账号和密码，输入即可
  • Step8:查看修改
    • 返回到github页面，可以查看到提交内容
    • 同步完成
  • Step9:删除提交
    • 本地文件删除提交更新到github中
    • 命令执行顺序：cd cloudsail—>git add .—>git commit -am "删除"—>git push -u origin master
    • 删除更新完成
  • Step10:其他常用命令
    • git push origin master //把本地源码库push到Github上
    • git pull origin master //从Github上pull到本地源码库
    • git config --list //查看配置信息
    • git status //查看项目状态信息
    • git branch //查看项目分支
    • git checkout -b host//添加一个名为host的分支
    • git checkout master //切换到主干
    • git merge host //合并分支host到主干
    • git branch -d host //删除分支host
