Alert/Confirm重写

  • Js alert/comfirm重写(外部文件引入：JQuery1.9及以上,bootstrap:bootstrap.css,bootstrap.js)
Js alert重写
函数名:alertwin
参数名 示例
message "添加成功"
callback 回调函数
调用示例
1个参数:Utils.alertwin("添加成功")
2个参数:Utils.alertwin('添加成功',function(){alert("我是回调函数");})
Js confirm重写
函数名:comfirmwin
参数名 示例
message "你确定要删除吗"
callback 回调函数
调用示例
Utils.comfirmwin('你确定要删除吗',function(){alert("我是回调函数");})
